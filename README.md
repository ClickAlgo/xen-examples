# Xen AI User Prompt Examples

Official example prompts for **Xen AI**, designed to help traders and developers build professional **cTrader cBots, indicators, dashboards, trading panels and plugins** using plain English.

Whether you are new to algorithmic trading or an experienced C# developer, these examples provide a starting point that you can load into Xen, customise and extend.

---

## About Xen AI

Xen AI is an AI-powered development environment built specifically for cTrader.

Unlike generic AI coding assistants, Xen understands the cTrader Automate API and uses structured workflows, guardrails and specialised prompts to generate higher quality trading tools.

Features include:

* Build complete cTrader cBots.
* Create custom indicators.
* Design interactive trading panels.
* Build professional dashboards.
* Convert strategies from MetaTrader and Pine Script.
* Repair and improve existing projects.
* Generate complete C# source code.
* Support for multiple AI models.

## How to Use

1. Open Xen AI.
2. Select the appropriate task.
3. Open the Prompt Library or copy a prompt from this repository.
4. Modify the prompt if required.
5. Submit the request.
6. Build and test the generated project inside cTrader.
